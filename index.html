How Encryption Works

All messages are encrypted end‑to‑end using the Signal Protocol (AES-256 in GCM mode with ECDH key exchange). Perfect forward secrecy ensures that even if a key is compromised, past messages remain secure.

Keys are generated locally on your device and never transmitted. Server stores only encrypted ciphertext. Participants verify each other's identities via out‑of‑band QR codes or key fingerprints.

Group chats use sender keys for efficient encryption, with each message uniquely encrypted for the group. No metadata is logged beyond required delivery information.
